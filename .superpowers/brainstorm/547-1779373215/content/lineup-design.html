How it works

  • Orbit screen stays exactly as-is on mobile (the animated solar system scales naturally)
  • Tapping the sun triggers the same veil + lineup transition — but vertical instead of horizontal
  • Sun anchors to top-center as the "← Orbit" back button
  • 8 planet rows stack below, each showing the planet canvas + section name + planet name
  • Column is scrollable if it overflows (unlikely on most phones, all 8 fit in ~560px)
  • Planets animate from orbit positions into rows with the same cubic-bezier as desktop
  • Tapping a row navigates to that section — same veil transition as desktop
